# Day 2 – Task 4: 4-Bit Universal Shift Register (USR) Design and Verification

## Overview

This project presents the design and verification of a **4-bit Universal Shift Register (USR)** using **Verilog HDL**. The design was implemented and simulated in **Xilinx Vivado** to validate its functionality across different operating modes.

A Universal Shift Register is a flexible sequential circuit that combines multiple data-handling capabilities within a single design. It can retain stored data, shift data in either direction, and load data in parallel, making it a fundamental component in modern digital systems.

## Objective

* Design a 4-bit Universal Shift Register using Verilog HDL.
* Implement multiple operating modes including Hold, Shift Left, Shift Right, and Parallel Load.
* Verify functionality through simulation and RTL analysis in Vivado.

## Applications

Universal Shift Registers are widely used in:

* Serial-to-Parallel Data Conversion
* Parallel-to-Serial Data Conversion
* Temporary Data Storage
* Digital Communication Systems
* Signal Processing Applications
* Embedded and Microprocessor-Based Systems
* Data Transfer and Synchronization Circuits

## Theory

A Universal Shift Register integrates the functionality of several types of shift registers into a single hardware structure. Depending on the selected control mode, the register can preserve existing data, shift bits left or right, or load multiple bits simultaneously.

## Functional Description

### Hold Mode (00)

The register maintains its current contents without modification.

```text id="qehq9s"
Q(next) = Q(previous)
```

### Shift Right Mode (01)

Data moves toward the least significant bit (LSB), while new serial data enters from the most significant bit (MSB) side.

```text id="zjk6lj"
Q3 ← Serial Input
Q2 ← Q3
Q1 ← Q2
Q0 ← Q1
```

### Shift Left Mode (10)

Data moves toward the most significant bit (MSB), while serial data enters through the least significant bit (LSB).

```text id="jofprf"
Q3 ← Q2
Q2 ← Q1
Q1 ← Q0
Q0 ← Serial Input
```

### Parallel Load Mode (11)

All register bits are updated simultaneously using the parallel input data.

```text id="77rxry"
Q ← p_in
```
