# Day 2 – Task 2: Synchronous SR Flip-Flop Design and Verification

## Overview

This project focuses on the design and verification of a **Synchronous SR (Set-Reset) Flip-Flop** using **Verilog HDL**. The circuit was simulated and analyzed using **Xilinx Vivado** to validate its behavior under different input conditions.

An SR Flip-Flop is a basic sequential storage element capable of holding a single bit of information. Unlike combinational circuits, its output depends not only on the current inputs but also on its previous state, making it an essential component in memory and control applications.

## Objective

* Design a synchronous SR Flip-Flop using Verilog HDL.
* Verify its operation through simulation.
* Analyze the synthesized RTL structure generated by Vivado.

## Applications

SR Flip-Flops are commonly used in:

* Digital Registers
* Binary Counters
* Memory Storage Elements
* Finite State Machines (FSMs)
* Control and Timing Circuits
* Sequential Logic Systems

## Theory

The SR Flip-Flop operates based on two control inputs:

* **S (Set)** – Forces the output HIGH.
* **R (Reset)** – Forces the output LOW.

The outputs are:

* **Q** – Main output
* **Q̅ (Q Bar)** – Complement of Q

The flip-flop updates its state only on the active edge of the clock signal.

### Truth Table

| S | R | Q(next)     | Q̅(next)     | Function |
| - | - | ----------- | ------------ | -------- |
| 0 | 0 | Q(previous) | Q̅(previous) | Hold     |
| 0 | 1 | 0           | 1            | Reset    |
| 1 | 0 | 1           | 0            | Set      |
| 1 | 1 | Undefined   | Undefined    | Invalid  |

## Operating Modes

### Hold Condition (S = 0, R = 0)

When both inputs remain LOW, the flip-flop preserves its previously stored value.

```text id="2lhjmb"
Q(next) = Q(previous)
```

### Reset Condition (S = 0, R = 1)

A HIGH Reset input clears the stored value.

```text id="6g0el7"
Q = 0
Q̅ = 1
```

### Set Condition (S = 1, R = 0)

A HIGH Set input stores a logic HIGH.

```text id="3q7xv0"
Q = 1
Q̅ = 0
```

### Invalid Condition (S = 1, R = 1)

When both Set and Reset are asserted simultaneously, the circuit enters an undefined state.

```text id="56dlmz"
Q = X
Q̅ = X
```

This condition is generally avoided in practical designs.

## Design Implementation

The SR Flip-Flop was implemented as a clocked sequential circuit in Verilog HDL.

### Inputs

| Signal | Description  |
| ------ | ------------ |
| clk    | System Clock |
| rst    | Reset Signal |
| s      | Set Input    |
| r      | Reset Input  |

### Outputs

| Signal | Description       |
| ------ | ----------------- |
| q      | Primary Output    |
| qbar   | Complement Output |

### Functional Description

At every active clock edge, the flip-flop evaluates the Set and Reset inputs and performs one of the following operations:

* Stores the previous value (Hold)
* Sets the output HIGH
* Resets the output LOW
* Produces an undefined state for invalid input combinations

The reset signal initializes the circuit to a known state before normal operation begins.

## Design Features

* Positive-edge triggered operation
* Synchronous state transitions
* Set and Reset control functionality
* Data retention during Hold condition
* Complementary outputs
* Sequential memory behavior
